# Hospital Analysis

A Hospital Data Analysis project built using Python and Pandas.

## Features

- Total number of patients
- Male and Female patient count
- Average patient age
- Oldest patient
- Youngest patient
- Department-wise analysis
- Most common disease in each department
- Average consultation fee

## Technologies Used

- Python
- Pandas

## Files

- hsptl_analysis.py
- patients.csv
- doctors.csv
- appointments.csv
